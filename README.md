# 🕌 Ajman Prayer Reminder (v1.0.0)
A modern, lightweight Windows desktop application that provides accurate prayer times for Ajman, UAE. Featuring a sleek UI, system tray integration, and automatic update tracking.

## ✨ Features
* **Real-time Countdown:** Always know how much time is left until the next prayer.
* **Smart Notifications:** Get desktop alerts for every prayer (Fajr, Dhuhr, Asr, Maghrib, Isha).
* **5-Minute Warning:** Optional beep to remind you to prepare for prayer.
* **Highly Customizable:** Pick your own accent colors and header gradients.
* **System Tray Support:** Close the window and let it run quietly in the background.
* **Auto-Update:** Notifies you instantly when a new version is released on GitHub.

## 🚀 How to Install
1. Go to the **[Releases](https://github.com/limonisyummy/Islamic-Prayer-Reminder/releases)** page.
2. Download `prayer_reminder.exe`.
3. Double-click to run! (No installation required).

## 🛠 Built With
* [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter) - Modern UI components.
* [Aladhan API](https://aladhan.com/prayer-times-api) - Precise timing data.
